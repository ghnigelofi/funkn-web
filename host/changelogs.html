- Bugs fixed
⟩ Previous bugs are now fixed. New bugs may appear. IDK.
 
- Updated UI
⟩ UI is now cleaner than ever. Messy channels look ugly, Funkn doesn't want that.

- New Feature!
⟩ You can now play your Spotify playlists, then Funkn. will find its equivalent on YouTube (can't fetch songs from Spotify directly for reasons)
Talk about that damn convenience!
 
- Bugs fixed
⟩ You can now use the SoundCloud integration again. Unlike Spotify, our SoundCloud integration is direct.
 
- New Design!
⟩ Just look at that fancy new profile picture.
 
Update v3.1 // 12-12-2020
- New Commands
⟩ You can now play ConnectFour with other members using Funkn.
 
- Overall Functionality Updated

⟩ You can now customize the prefix on your server.

Update v2.7a // 7-6-2020
- Funkn. is now sharded!
⟩ Funkn. should now give better music quality and performance.
 
- Better Statistics
⟩ You can now check Funkn.'s RAM, Memory, and your server's shard number.
 
- Fixed bugs
⟩ Previous bugs were leaving the channel unexpectedly, music timeouts, and etc.
 
Update v2.6p // 7-4-2020

Update v2.3 // 7-1-2020
- Progress Bar!
⟩ You can now see the progress in the nowplaying command.
