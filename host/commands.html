Commands List

 
The command list is updated every major update.

A complete list of Funkn's commands.
Aliases also trigger commands, for ease.
 

All Commands
 
Command Description Usage Aliases
play Plays the first track it finds on YouTube. f.play [song] p
search search 10 songs, and pick which of them will play. f.search [song] find f
skip skips the current song playing. f.skip s
skipto skip to a certain song in queue. f.skipto [queue #] st jump
nowplaying sends the details of the current song playing. f.nowplaying np
queue returns the song queue for the server. f.queue q
remove removes a song from the queue. f.remove [queue no.] r
clean deletes 100 messages from Funkn. f.clean purge
move move a song's position in the queue. f.move [# of song to be moved] [new position] m
removerange remove a number of songs in the queue. f.removerange [remove start #] [remove end #] rr
pause pauses the current song playing. f.pause pau
resume resumes the current paused song. f.resume res
join make Funkn join the voice channel you're in. f.join j
disconnect disconnect Funkn from a voice channel. f.disconnect dc leave
volume check, raise, or decrease the volume. f.volume [number] vol
lyrics get lyrics from a song. f.lyrics [song] ly
clear clear the whole song queue for the server. f.clear None
loop loops the current track. f.loop lp
duplicate duplicates the song once. f.duplicate dupe
ping checks the API and Bot latencies. f.ping pp
replay replays the current song playing. f.replay rep
spotify fetches your Spotify user presence. f.spotify listening
shuffle shuffle the queue made. f.shuffle sh shuf
bassboost increase or decrease the bass. f.bassboost [new bass number] status
help if you need help, call this command. f.help h commands
invite invite Funkn to your server. f.invite None
24h enable or disable 24 hour mode. Always free. f.24h 24H h24
playlist add, delete, or wipe clean your playlist library. ONLY YouTube playlists work! f.playlist [add | delete | reset | library] pl savedpl
prefix changes Funkn's server prefix.' f.prefix [new_prefix] sp setprefix
